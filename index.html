toneSampler

Room tone regeneration plugin for Pro Tools AudioSuite

toneSampler was created in partial fulfillment of my NYU Music Technology thesis. It is truly a labor of love, and something I've dreamed of creating since I first began my film audio career. Room tone is one of the most tedious tasks of any film audio editor. My goal was to solve this issue by creating a novel algorithm that samples and resynthesizes unique room tone, to be rendered by the user endlessly.


toneSampler is currently in the alpha stage. Please check back to see updates!

8-Bit Machine

8-Bit Machine

The Zonkey™️ 8-Bit Machine is one of my first audio software projects, written in Max MSP. In middle school I dreamed of being a famous chiptune producer, which inspired me to make a standalone chiptune creator. It features an 8-Bit style synthesizer, a bitcrushed drum machine, an "improv" algorithm, and more. It's not the most serious creation, but I've spent hours playing around with the sequencer.

Hardware Instrument Design.

I'm no physicist, but I love to play around with circuits to make weird DIY instruments.



Homemade Spring Reverb

I built a spring reverb using an old portable iPod speaker and a piezo microphone. My grandfather and I built a box, using sawed pine branches as knobs. It's not studio quality, but it has a very unique sound.



♥︎ Heart Synth ♥︎

I built a heart-controlled digital synthesizer/step sequencer. It was built using a heart-rate monitor that can be worn on a finger. The signal analysis and synthesis both happen on an Arduino Teensy microcomputer.



'Sawed-Off Shorgan'

This one's a bit strange - using an old bucket, I built an acoustic pipe synthesizer. Technically the pipe is not the actual synthesizer, merely a filter. Max MSP controls three speaker cones, which output white noise into each pipe. The result is three distinct pitched percussive sounds.
